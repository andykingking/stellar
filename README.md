# Stellar

A convenient way to get a relevant background for your desktop.

## Installation

  1. Clone the repository:

    ```bash
    git clone git@github.com:andykingking/stellar.git
    cd stellar
    ```

  2. Install dependencies:

    ```bash
    mix deps.get
    ```

  3. Build the application:

    ```bash
    mix escript.build
    ```

  4. Run the application:

    ```bash
    ./stellar <filename>
    ```
